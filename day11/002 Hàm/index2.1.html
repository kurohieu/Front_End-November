Hàm trong js
        Hàm trong js ( function )
        Function là 1 tập hợp các câu lệnh được xây dựng
        để thực hiện 1 nhiệm vụ nào đó
        ví dụ
        Function tính chu vi hình tròn
        Function tính bán kính hình tròn
        Function tính tiền nước
        Thay vi phải viết rất nhiều dòng code lặp đi lặp lại để thực hiện
        1 chức năng người ta sẽ tạo ra 1 function để thực hiện chức năng đó

        Function sẽ đầu vào input và đầu ra output
        đầu vào chính là các tham số của hàm
        đầu ra thường sẽ được trả về bởi lệnh return
        lệnh return sẽ kết thúc hàm

        cú pháp
        function ten_ham(tham_so_dau_vao_1,tham_so_dau_vao_2,tham_so_dau_vao_3) {
            //  các câu lệnh nội dung của hàm

            // return trả về kết quả nếu cần;
        }
